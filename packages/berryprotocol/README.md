# berryprotocol

Facade package for the Berry SDK.

## Install

```bash
npm install berryprotocol
```

## Usage

```ts
import BerryProtocol from "berryprotocol";
```

For OTP flows, install the official companion package:

```bash
npm install @berrysdk/berry-otp
```

```ts
import { BerryOTP } from "@berrysdk/berry-otp";
```
