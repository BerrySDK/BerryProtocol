# Getting Started

## Install

```bash
npm install
cp .env.example .env
```

## Run

```bash
npm run dev
```

## Default URLs

- `GET http://localhost:3000`
- `GET http://localhost:3000/info`
- `GET http://localhost:3000/docs`

## Authentication

Use:

```http
Authorization: Bearer your_api_key
```

The development default is:

```text
berryapi_dev_key
```
