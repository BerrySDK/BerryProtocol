# BerryAPI

BerryAPI is the BerrySDK automation layer for WhatsApp Web, powered by BerryProtocol.

It gives you a clean REST surface for:

- `n8n`
- `Typebot`
- internal dashboards
- webhook integrations
- realtime WhatsApp event streams

## Core Features

- Multi-instance management
- QR and pairing-code connection flows
- Modern message endpoints
- Webhook dispatching
- WebSocket realtime events
- Swagger docs at `/docs`
- SQLite by default
